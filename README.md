I build robots that learn to navigate uncertain environments. My research combines **meta-reinforcement learning** with **adaptive control** to create systems that generalize across conditions they've never seen—from varying terrain dynamics to GPS-denied navigation.

### Research Focus

**Adaptive Control & Learning** — Developing domain-randomized RL policies (PPO, SAC) for differential-drive robots that achieve >96% mission completion across friction coefficients from 0.1μ to 1.0μ, with <6cm cross-track error.

**Resilient Localization** — Designed the [SHARF algorithm](https://ieeexplore.ieee.org/document/10679455) for GPS spoofing defense. Built [SatTrack](https://isabelmoore.github.io) with custom Kalman filtering for smooth 60fps orbital visualization.

**AV Perception Systems** — Integrated monocular depth estimation (Depth Anything, SAM2) into the [Mcity Data Engine](https://github.com/mcity/mcity_data_engine) for iterative model improvement.
