👋 Hello! I’m **Isabel Moore**, a Master’s student in Computer Engineering at Texas A&M University. I work on autonomous vehicles, cybersecurity, and resilient real-time networking in the [CAST Lab](https://cast.engr.tamu.edu) and the Bush Combat Development Complex (BCDC). I’m passionate about combining meta-reinforcement learning with adaptive control.

## 👩‍💻 Projects:
- **Mcity 2.0 Demo:** Depth & segmentation integration for autonomous vehicles  
- **Mcity Data Engine:** A data-centric search engine for AV perception—smart sampling, labeling, and anomaly detection; integrates Voxel51, CVAT.AI, Hugging Face, Ultralytics, Weights & Biases, and Anomalib  
- **Cyber-Physical Security:** Research on secure localization and resilient network architectures  
- **Meta-RL Differential Drive Model:** Training a Dynamic Differential-Drive (DDD) model in Gazebo simulation using Meta Reinforcement Learning
